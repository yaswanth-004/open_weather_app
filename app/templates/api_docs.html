📖 API Documentation

Overview

WeatherApp uses two external APIs and exposes internal routes for weather data.

External APIs Used

1. Nominatim (OpenStreetMap)

Purpose: Convert city/country/ZIP into latitude & longitude.

Endpoint: https://nominatim.openstreetmap.org/search

Method: GET

Returns: List of location objects with lat/lon.

Rate limit: 1 request/second (free, no key needed).

2. OpenWeatherMap

Purpose: Fetch current weather and 5-day forecasts.

Current Weather

Endpoint: https://api.openweathermap.org/data/2.5/weather

Params: lat, lon, appid, units=metric

Returns: Temperature, humidity, wind, pressure, sunrise/sunset, icon.

5-Day Forecast

Endpoint: https://api.openweathermap.org/data/2.5/forecast

Params: lat, lon, appid, units=metric, cnt=40

Returns: 40 readings (every 3 hours for 5 days).

Free tier limit: 60 calls/minute, 1,000,000 calls/month.

Rate Limiting

Each user gets 20 free API calls per day.

Formula: capacity = total_users × 20

If your calls_today ≥ 20, you are redirected to the rate limit page until midnight UTC.

Multiple frontend requests (today + forecast) each count as 1 API call.
